# Friendship Quiz - Supabase Responses

This version saves quiz answers to Supabase (works with GitHub Pages).

## 1) Create a Supabase project

1. Go to https://supabase.com and create a new project.
2. In **Project Settings → API**, copy:
   - **Project URL**
   - **anon public key**

## 2) Create the table

Run this SQL in the Supabase SQL editor:

```sql
create table if not exists responses (
  id bigint generated by default as identity primary key,
  name text,
  answers jsonb not null,
  answered_count integer not null,
  questions jsonb not null,
  submitted_at timestamptz not null default now()
);
```

## 3) Add your Supabase keys

Open `script.js` and replace:

```javascript
const SUPABASE_URL = "PASTE_YOUR_SUPABASE_URL_HERE";
const SUPABASE_ANON_KEY = "PASTE_YOUR_SUPABASE_ANON_KEY_HERE";
```

## 4) Enable inserts (Row Level Security)

In Supabase:

1. Go to **Table Editor → responses → RLS** and enable RLS.
2. Add this policy to allow inserts from the public app:

```sql
create policy "Allow public inserts"
on responses
for insert
to anon
with check (true);
```

## 5) Publish to GitHub Pages

After this, every submission will appear in your `responses` table.
